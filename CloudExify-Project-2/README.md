## 📌 Overview

A command-line grade management system for teachers to manage student records, enter grades for 5 subjects, calculate averages, generate pass/fail reports, and save data persistently to CSV.

---

## 🎯 Features

- ✅ Add students with grades for 5 subjects
- ✅ View all students in a formatted table
- ✅ Class report with rankings (🥇🥈🥉) and stats
- ✅ Search student by name
- ✅ Edit grades for any subject
- ✅ Delete student with confirmation
- ✅ Auto-creates and auto-saves to CSV
- ✅ Data persists across sessions
