## 📌 Overview

A command-line expense tracking application built in Python. Users can add, view, filter, and delete expenses by category. All data is stored persistently in a JSON file.

---

## 🎯 Features

- ✅ Add new expense with description, amount, and category
- ✅ View all expenses in a formatted table
- ✅ View total spending by category with percentages
- ✅ Filter expenses by category
- ✅ Delete expense by ID with confirmation
- ✅ Data persists using JSON file storage
- ✅ Input validation for amount (positive numbers only)
- ✅ Error handling for invalid inputs

---

## ⚙️ Tech Stack

- Python 3.x
- JSON (file storage)
- Command Line Interface
